# Verilog Projects

A repository showcasing digital logic design implementations in Verilog and SystemVerilog.

## Contents
* **[Synchronous FIFO](./sync_fifo/)**: A parameterized synchronous FIFO buffer with full, empty, almost_full, almost_empty, overflow, and underflow flag indicators.
* **[Counter Designs](./counter/)**: A collection of counter implementations including a basic 4-bit Up/Down counter and a spec-compliant, low-power 32-bit counter with handshake controls.
