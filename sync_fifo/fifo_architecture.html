Synchronous FIFO RTL Architecture

Hardware block diagram representing data paths, pointers, and control flag generation

I/O Pins & Interfaces

  • • wdata[7:0]: Input bus to write data.
  • • rdata[7:0]: Output data register (synchronized).
  • • wr_en & rd_en: Read and write request strobes.
  • • clk & rst: Global clock & synchronous reset.

Pointers & Control Logic

  • • Pointers (5-bit): Extended pointers detect lap states.
  • • Address Mapping: Lower 4 bits index the 16 slots.
  • • Pointer Subtraction: Current element occupancy computed dynamically to flag almost states.

Flag & Error Systems

  • • Empty/Full: Combinational flags derived from comparing pointers.
  • • Almost levels: Configurable threshold flags.
  • • Overflow/Underflow: Sticky alert registers triggered on violation commands.
